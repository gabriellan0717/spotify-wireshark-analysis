# Spotify Wireshark Network Analysis

A detailed network analysis project using **Wireshark** to capture and inspect real-time internet traffic generated by the Spotify desktop application. The goal was to understand how a modern streaming service communicates over the network at the packet level.  

**Completed for ITEC 250 — Towson University**

## Project Highlights

- Captured **8,885 packets** over a 50-second session of Spotify usage.
- Analyzed packet-level details including **protocols, encryption, and server infrastructure**.
- Traced network paths to Spotify CDN servers to understand data routing.
- Demonstrated **real-time analysis** skills using industry-standard tools.
- Applied networking knowledge from **ITEC 250**, covering TCP/IP, DNS, and HTTPS traffic.

---

## Key Takeaways

- Modern streaming services like Spotify use **encrypted HTTPS traffic** on dynamic ports to securely deliver content.
- Tools like Wireshark provide insights into **network behavior**, including server endpoints, packet sizes, and communication patterns.
- Understanding network traffic is crucial for **network administration, security, and performance monitoring**.
