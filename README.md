# spotify-wireshark-analysis
Wireshark packet capture and analysis of live Spotify traffic, protocols, encryption, and server infrastructure.

Spotify Network Traffic Analysis 🔬
A network analysis project using Wireshark to capture and inspect real-time internet traffic generated by the Spotify desktop application. The goal was to understand how a modern streaming service communicates over the network at the packet level.
Completed for ITEC 250 — Towson University
